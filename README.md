# Ziya Ata Yazici - Photography Portfolio

A clean, modern photography portfolio website showcasing my work as a photographer and researcher. This minimalist portfolio focuses purely on visual storytelling through a curated collection of photographs.

## About

This website serves as a focused digital portfolio featuring:
- **Photography Gallery**: A curated collection of my photographic work with interactive lightbox viewing

## Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Modern UI**: Clean, minimalist single-page design with smooth animations
- **Lightbox Gallery**: Interactive photo viewing experience with Fancybox
- **Dark/Light Mode**: Automatic theme switching based on user preference
- **Fast Loading**: Optimized performance with modern web technologies
- **Minimalist Approach**: Focus purely on the photography without distractions

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Alpine.js
- Fancybox (for image gallery)

## Live Website

Visit my portfolio at: https://yaziciz.github.io
